# 移动充电服务应用

一个便捷的移动充电服务应用，提供充电站查找、预约、充电状态监控等功能。

## 功能特点

- 🗺️ 附近充电站查找与地图显示
- 🔍 智能搜索和筛选功能
- 📋 充电站预约服务
- ⭐ 智能推荐系统
- 👥 社区评价与分享
- 📱 响应式设计，适配各种设备

## 技术栈

- HTML5
- CSS3
- JavaScript (ES6+)
- 模拟API服务

## 在线访问

本项目部署在Netlify上，可通过以下链接访问：
[https://your-app-url.netlify.app](https://your-app-url.netlify.app)

## 本地运行

1. 克隆仓库到本地
2. 在项目目录中启动本地服务器：
   ```bash
   python -m http.server 8000
   ```
3. 在浏览器中访问 `http://localhost:8000`
